Ioctl Command Code Reference

All ioctl command codes understood by the datadev driver and associated headers. Pass these codes as the second argument to ioctl(fd, cmd, arg).

Note

The DMA_Ret_Index command encodes a count in the upper 16 bits of the command code. To return N buffers, use DMA_Ret_Index | ((N << 16) & 0xFFFF0000). For a single index, use DMA_Ret_Index | 0x10000.
